# ecommerce-iac-aws

Plataforma de e-commerce escalable desplegada en AWS con Infrastructure as Code.
**Proyecto Final — Infraestructura III**

## Stack
- React 18 + Vite (frontend)
- Spring Boot 3 + Java 21 (backend)
- PostgreSQL en AWS RDS
- CloudFormation para toda la infraestructura

## Prerrequisitos
- AWS CLI configurado con credenciales de sandbox
- Java 21 + Maven
- Node.js 20+
- cfn-lint (`pip install cfn-lint`)
